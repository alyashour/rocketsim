# Style Guide

## Headers

Only use header guards (no pragma hooliganism).
Project name is RocketSim so prefix with RS.
```c
#ifndef RS_MY_MODULE_H
#define RS_MY_MODULE_H

#endif
```

## Naming

Variables: camelCase  
Functions: cascalCase  
Structs: PascalCase  

## Organization
Put .c and .h files in the same directory.
